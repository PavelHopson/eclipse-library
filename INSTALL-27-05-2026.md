### 2️⃣ knowledge-work-plugins (Anthropic) → Eclipse AI Hub

**Где:** `eclipse-ai-hub/.claude/plugins/knowledge-work-plugins/` (git clone --depth 1)

**Что внутри (11+ plugins):**
- `bio-research` — биоресёрч
- `customer-support` — клиентская поддержка
- `data` — анализ данных, дашборды
- `design` — дизайн pipelines
- `engineering` — инженерные задачи
- `finance` — финансовая аналитика
- `human-resources` — HR onboarding
- `enterprise-search` — поиск по корпоративным данным
- `cowork-plugin-management` — мета (plugin manager)

**Как использовать:**
```bash
# В Claude Code сессии открытой в eclipse-ai-hub/
/plugin marketplace add ./.claude/plugins/knowledge-work-plugins
/plugin install finance@knowledge-work-plugins
/plugin install design@knowledge-work-plugins
# и так далее по модулям Hub'а
```

Каждый plugin = independent skill collection, можно cherry-pick под конкретный
Hub module (например, finance plugin → Hub Copywriter «Финансовый отчёт» preset).
