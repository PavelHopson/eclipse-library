# Веб-утилиты — кураторский каталог

> Объединённый каталог из двух подборок Telegram «Не баг, а фича» (35 сайтов — 14.05.2026; 50 сайтов — 21.05.2026). Дубликаты слиты, всё разбито по задачам. ~80 сервисов.
>
> Все ресурсы бесплатные или с заметным free-tier на момент сохранения — условия меняются, проверяйте перед использованием в продакшне.
>
> **⚠️ Grey-zone / пиратские ресурсы.** Часть инструментов даёт доступ к платному или копирайт-контенту в обход правообладателя (12ft.io, libgen, Sci-Hub, archive.ph для свежих статей). В большинстве юрисдикций это нарушение авторского права. Помечены ниже значком 🔓. Используем для личного ресёрча на свой риск — **не** для коммерческой репликации платного контента и **не** в клиентских продуктах Eclipse.
