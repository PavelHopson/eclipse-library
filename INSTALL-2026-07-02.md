## reference

### ZCode / GLM-5.2

- Уже есть в library как provider/model radar.
- Дополнительно смотреть UX: long-running tasks, mobile/messenger control, GLM-optimized coding flow.
- Не принимать маркетинг "Claude Code killer" без собственных задач и evals.

### GFusion / Sber diffusion LLM

- Держать в model radar.
- До интеграции найти официальный repo, license, weights, inference recipe, hardware needs and evals.

### caveman

- Уже есть в library как cost-control.
- Использовать только для agent loops, где краткость важнее стиля.
- Не применять для клиентских текстов, UX copy, юридических/финансовых ответов.

## hold / do-not-use

### edu mail benefits navigator

- Не использовать и не рекомендовать способы получить student email, если человек не студент.
- Риск: fraud, ToS violation, блокировки аккаунтов и репутационный ущерб.
- Легитимно: только официальный student/education access при реальном праве на него.

### Fingerprint / privacy tools - caution

- Не ставить массово "всё из списка".
- Каждое расширение и утилиту проверять отдельно: source, permissions, maintainer, release activity.
