# Resume Toolkit — 7 промптов для резюме и найма

> Источник: Telegram «Не баг, а фича» 04.06.2026. Полный путь: разбор
> слабых мест → обход ATS → переписывание опыта в результаты → summary →
> адаптация под вакансию → сопроводительное → симулятор собеседования.
>
> **Куда применяем:** связка с уже залистанными [job-ops](https://github.com/DaKheera47/job-ops)
> (авто-поиск работы + ATS-обход) и [Eclipse AI Hub Copywriter](https://github.com/PavelHopson/eclipse-ai-hub)
> — можно встроить как пресеты «Резюме / Карьера». ATS-оптимизация (`#2`, `#5`)
> и STAR-симулятор (`#7`) — самые ценные. Промпты ждут вставки текста в `[...]`.

---

## 1️⃣ Анализ причин отказа (беспощадный аудит)
```
Проанализируй это резюме [вставь] и перечисли 10 причин, по которым HR
отвергнет его за 6 секунд. Для каждой: где ошибка, почему это повод для отказа,
исправленная версия. Отсортируй от самой опасной к наименее опасной. Будь беспощаден.
```

## 2️⃣ Оптимизация под ATS-фильтр
```
Моё резюме [вставь] должно пройти ATS-фильтр для вакансии [вставь]. Найди все
недостающие ключевые слова, укажи, в какой части резюме их естественно добавить,
и оцени совместимость до и после правок по шкале 0–100.
```

## 3️⃣ Опыт → измеримые результаты
```
Возьми рабочий опыт из резюме [вставь] и перепиши каждый пункт: вместо описания
задач — результат (мощный глагол + конкретное достижение + цифра/влияние).
Покажи «до/после» рядом. Если цифр нет — помоги их оценить правильными вопросами.
```

## 4️⃣ Цепляющее summary (5 версий)
```
Напиши 5 версий профессионального summary. Данные: [Должность, Годы опыта,
Область, Достижения, Цель]. Каждая — 3 строки: позиция + конкретный результат +
карьерная ориентация. Без клише «амбициозный профессионал». Цель: чтобы
читатель сразу сказал «хочу познакомиться».
```

## 5️⃣ Адаптация под конкретную вакансию
```
Адаптируй резюме [вставь] под вакансию [вставь]. Добавь ключевые слова для ATS,
перепиши пункты опыта под требования, сделай summary под целевую позицию, удали
лишнее. Дай финальную версию, готовую к отправке.
```

## 6️⃣ Сопроводительное письмо, которое прочитают
```
Проанализируй вакансию [вставь] обратным разбором и определи 3 критические
проблемы/потребности компании. Напиши сопроводительное на 200 слов, сопоставив
мой опыт [вставь] с каждой из них: цепляющее введение, конкретные доказательства
результатов, чёткий призыв к собеседованию.
```

## 7️⃣ Симулятор собеседования (метод STAR)
```
Проведи собеседование на должность [Название] в компании [Название]. Задавай
10 вопросов постепенно — поведенческие, технические, сценарные. Я отвечаю, ты
оцениваешь каждый ответ 0–10, объясняешь слабые места и показываешь, как
переформулировать через STAR. В конце — комплексная оценка и план развития.
```

---

**Паттерн:** `#1` (аудит) → `#3` (опыт в результаты) → `#5` (адаптация под
вакансию) → `#6` (сопроводительное) → `#7` (репетиция собеседования).
