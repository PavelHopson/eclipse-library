# 📄 Resume-boost промпты (3) — подгонка CV под вакансию

> Источник: Telegram «Не баг, а фича», 15.06.2026. Три коротких промпта, чтобы пройти
> ИИ-скрининг и попасть на живой собес. **Дополняют** более полный
> [resume-toolkit-7.md](resume-toolkit-7.md) (7 промптов по карьере).
>
> Использование: загрузите в LLM исходное резюме + текст вакансии, затем по очереди.
> **Честно:** правьте под **реальный** опыт — выдуманные достижения вскроются на собесе.

**Куда внедрить:** [InterviewForge](https://github.com/PavelHopson/InterviewForge) (career-копилот),
[Eclipse AI Hub](https://github.com/PavelHopson/eclipse-ai-hub) Copywriter, личный workflow.

---

### 1. Подстраиваем резюме под вакансию (gap-анализ)
```
Верни мне топ-3 пробела между моим CV и этой вакансией, и закрой их, используя только тот
опыт, что у меня есть.
```

### 2. Редактируем summary под ключевые слова
```
Перепиши summary в три строки так, чтобы был match восемьдесят процентов ключевых слов
из описания вакансии, но без keyword-stuffing.
```

### 3. Достижения с цифрами
```
Придумай три achievement-bullets под каждую роль. Конкретные. С цифрами. На основе моего
реального опыта и того, что компания ищет.
```
