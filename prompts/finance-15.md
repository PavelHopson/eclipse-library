# 💰 Финансовые промпты (15) — личные финансы

> Источник: Telegram «Не баг, а фича», 15.06.2026. Набор для личного бюджета —
> распределение зарплаты, долги, инвестиции, подушка, рост дохода.
> **Это не трейдинг** (в отличие от [CryptoPulse](https://github.com/PavelHopson/CryptoPulse)) —
> бытовое планирование. Подставляйте свои значения в `[СКОБКИ]`.
> **Дисклеймер:** вывод LLM — не финансовый совет; для крупных решений сверяйтесь со специалистом.

**Куда внедрить:** пресет-персона «Финансовый стратег» в [Eclipse AI Hub](https://github.com/PavelHopson/eclipse-ai-hub) (Chat / Copywriter), личный workflow.

---

### 1. Стратег распределения зарплаты
```
Действуй как стратег по личным финансам. Моя ежемесячная зарплата составляет [ЗАРПЛАТА],
фиксированные расходы — [РАСХОДЫ], текущие сбережения — [СБЕРЕЖЕНИЯ], а моя финансовая
цель — [ЦЕЛЬ]. Создай умный план распределения средств между тратами, сбережениями и инвестициями.
```

### 2. Анализатор денежных потоков
```
Мой ежемесячный доход составляет [ЗАРПЛАТА], а расходы — [СПИСОК РАСХОДОВ]. Проанализируй,
как сейчас течёт мой бюджет каждый месяц, и предложи более эффективную структуру, которая
позволит значительно повысить норму сбережений.
```

### 3. План роста богатства на 5 лет
```
Мой доход составляет [ЗАРПЛАТА], я могу инвестировать [ЕЖЕМЕСЯЧНЫЕ ИНВЕСТИЦИИ] в месяц,
текущий чистый капитал — [ЧИСТЫЙ КАПИТАЛ]. Построй реалистичный 5-летний план значительного
роста моего капитала.
```

### 4. Стратегия избавления от долгов
```
Я зарабатываю [ЗАРПЛАТА] в месяц и имею долги на общую сумму [СУММА ДОЛГА] с процентными
ставками [ПРОЦЕНТНЫЕ СТАВКИ]. Создай самую быструю и эффективную стратегию полного избавления
от этих долгов.
```

### 5. Система инвестирования из зарплаты
```
Моя ежемесячная зарплата составляет [ЗАРПЛАТА], я хочу инвестировать [СУММА] каждый месяц.
Мой уровень риска — [НИЗКИЙ / СРЕДНИЙ / ВЫСОКИЙ]. Предложи простую долгосрочную
инвестиционную стратегию.
```

### 6. Расчёт времени до финансовой независимости
```
Мой ежемесячный доход составляет [ЗАРПЛАТА], расходы — [РАСХОДЫ], ежемесячные инвестиции —
[ИНВЕСТИЦИИ]. Оцени, сколько времени потребуется, чтобы достичь финансовой независимости,
и какие изменения помогут ускорить этот процесс.
```

### 7. Планировщик улучшения образа жизни
```
Я зарабатываю [ЗАРПЛАТА] и хочу улучшить качество жизни, продолжая наращивать богатство.
Предложи, как можно разумно увеличить расходы на комфорт, не замедляя финансовый прогресс.
```

### 8. План увеличения дохода за 90 дней
```
Мой текущий доход — [ЗАРПЛАТА]. Учитывая мои навыки [НАВЫКИ], доступное время [ЧАСЫ]
и рыночный спрос, предложи 3 реальных способа увеличить доход в ближайшие 90 дней
с чёткими шагами.
```

### 9. Преобразователь навыков в доход
```
Я владею следующими навыками: [НАВЫКИ]. Покажи, как превратить их в источники дохода.
Включи фриланс, цифровые продукты и масштабируемые варианты с указанием потенциальной прибыли.
```

### 10. Валидатор идей подработок
```
Предложи 5 идей подработок на основе моего профиля [НАВЫКИ + ИНТЕРЕСЫ]. Для каждой идеи
оцени спрос, стартовые усилия, время до первого дохода и потенциал масштабирования.
```

### 11. Оптимизатор расходов
```
Вот мой месячный бюджет расходов: [РАЗБИВКА РАСХОДОВ]. Найди лишние траты, скрытые утечки
и ненужные подписки. Покажи, как сократить расходы минимум на 20%, не ухудшая качество жизни.
```

### 12. Сборщик инвестиционного портфеля
```
Я могу инвестировать [СУММА] в месяц. Собери диверсифицированный портфель, включающий акции,
облигации и альтернативные активы. Укажи процентное распределение и ожидаемую долгосрочную доходность.
```

### 13. Симулятор умножения капитала в 10 раз
```
Мой текущий капитал — [ЧИСТЫЙ КАПИТАЛ], доход — [ЗАРПЛАТА], инвестиции — [ЕЖЕМЕСЯЧНЫЕ].
Просимулируй разные сценарии, которые позволят умножить капитал в 10 раз быстрее,
и сравни результаты.
```

### 14. Планировщик налоговой оптимизации
```
Мой годовой доход составляет [ДОХОД]. Предложи полностью легальные способы снизить налоги,
оптимизировать вычеты и максимально увеличить доход после налогообложения.
```

### 15. Архитектор финансовой подушки
```
Мои ежемесячные расходы составляют [РАСХОДЫ]. Построй пошаговый план создания полностью
сформированного фонда безопасности (emergency fund) и укажи, где его лучше хранить
с точки зрения ликвидности и безопасности.
```
