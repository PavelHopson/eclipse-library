# ✈️ Travel-hacking промпты (10) — дешёвые перелёты и поездки

> Источник: Telegram «PushEnter», 16.06.2026. Планирование маршрута, поиск дешёвых
> авиабилетов, отелей, ресторанов и экскурсий. Подставляйте города/даты в `[СКОБКИ]`.
>
> **⚠️ Юридическая рамка.** Часть приёмов — серая зона по правилам авиакомпаний:
> - **Hidden-city / «выход на пересадке» (#4)** и **mistake fares (#6)** нарушают условия
>   перевозки большинства авиакомпаний → риск аннуляции обратного билета, потери статуса/мили,
>   бана аккаунта, проблем с багажом. Использовать на свой риск, **не** для делового тревела.
> - **Раздельные билеты (#5)** — легально, но риск при сорванной стыковке (отдельные тарифы
>   не защищают связку).
>
> LLM не имеет доступа к live-ценам — промпты дают **стратегию и что проверять**, а не точные
> цены. Финальную бронь делать на агрегаторах (Skyscanner / Aviasales / Google Flights).

**Куда внедрить:** личный workflow; концепт будущего travel-ассистента. Прямого продукта-фита нет.

---

### 1. Выбираем время для покупки дешёвых билетов
```
Я лечу из [город A] в [город B] в период [месяц/сезон]. Проанализируй исторические паттерны
цен на авиабилеты по этому маршруту: за сколько недель до вылета цены обычно минимальны, есть ли
оптимальные дни недели для покупки и когда начинается рост цен. Дай конкретные сроки и процентный
выигрыш от покупки в правильный момент.
```

### 2. Ищем билеты по гибким датам
```
Мне нужно лететь из [A] в [B] примерно [дата], но я могу сдвинуть вылет на ±3 дня. Составь
таблицу: для каждой из 7 дат покажи ожидаемый диапазон цен, загруженность рейса и оптимальное
время суток для вылета. Укажи, какая дата даст максимальную экономию и почему.
```

### 3. Собираем ближайшие аэропорты вылета и прилёта
```
Я вылетаю из [город A] и лечу в [город B]. Найди все аэропорты в радиусе 150 км от каждого
города. Для каждой альтернативной пары аэропортов оцени: потенциальную экономию на билете,
стоимость и время наземного трансфера, итоговую выгоду. Выведи топ-3 комбинации по соотношению
цена/удобство.
```

### 4. Hidden-city: выходим посередине пути ⚠️
```
Объясни, как применить стратегию hidden city для маршрута [A] → [B]. Найди рейсы, где [B]
является транзитным городом на пути в [C], если они стоят дешевле прямого [A]→[B]. Укажи риски
(потеря багажа, аннуляция обратного билета, политика авиакомпании), на каких авиакомпаниях трюк
работает лучше, и как минимизировать вероятность отказа в посадке.
```
> ⚠️ Нарушает условия большинства перевозчиков — риск аннуляции/бана.

### 5. Разбиваем маршрут по частям
```
Маршрут [A]→[B] стоит [цена]. Разбей его на 2–3 отдельных сегмента через популярные хабы.
Для каждого варианта подели: итоговую стоимость всех сегментов, минимальное время стыковки,
какие авиакомпании/лоукостеры покрывают каждый отрезок. Предупреди о рисках раздельных билетов
и как их снизить.
```

### 6. Ловим mistake fares ⚠️
```
Перечисли все надёжные ресурсы, телеграм-каналы и сервисы для отслеживания ошибочных тарифов
и глючных цен на авиабилеты (mistake fares) для жителя [страна/город]. Для каждого укажи:
как часто выходят предложения, как быстро нужно реагировать, как правильно бронировать,
чтобы минимизировать риск отмены авиакомпанией.
```
> ⚠️ Ошибочные тарифы авиакомпания вправе аннулировать.

### 7. Меняем регион/валюту покупки
```
Для рейса [авиакомпания], маршрут [A]→[B], вылет [дата]: составь чеклист покупки билета с разных
региональных сайтов и в разных валютах. Какие страны исторически дают самые низкие тарифы на эту
авиакомпанию? Как безопасно оплатить с иностранной карты или через зарубежный платёжный метод?
Укажи, на каких маршрутах разница в цене максимальна.
```

### 8. Ищем крутые отели по низким ценам
```
Мне нужен отель в [город], [даты], бюджет до [сумма]/ночь. Найди варианты с рейтингом 8.5+
на Booking/Tripadvisor, которые при этом заметно дешевле аналогов — из-за нераскрученности,
удалённости от туристического центра или недавнего открытия. Для каждого укажи: реальное время
до центра, что включено в цену, скрытые сборы, и есть ли смысл бронировать напрямую у отеля для скидки.
```

### 9. Едим там, где едят местные
```
Я в [город] на [N] дней, интересует [кухня/тип еды] и [тип культуры — музеи, уличное искусство,
архитектура]. Составь список: 5 ресторанов/кафе, куда ходят местные, а не туристы (с ценой
среднего чека), и 5 культурных мест с реальной ценностью. Для каждого — одна причина, почему
именно оно, и одна конкретная деталь, которую нельзя пропустить внутри.
```

### 10. Планируем экскурсии сами вместо дорогих туров
```
Я в [город] на [N] дней, хочу увидеть максимум без переплаты за организованные экскурсии.
Составь маршрут по дням: сгруппируй достопримечательности по районам, чтобы минимизировать
перемещения. Для каждого дня укажи: бесплатные входы и часы, когда нет очередей, один
недооценённый объект рядом с туристическими, и местный транспорт вместо такси.
```
