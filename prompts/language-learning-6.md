# 🗣️ Изучение языка с нейросетью — 6 промптов (замена Duolingo)

> Шесть промптов превращают любую чат-нейросеть (Claude / Gemini / GPT / локальную)
> в персонального репетитора. Работают для **любого языка** — подставь нужный в
> `[ЯЗЫК]`. Источник: дроп Telegram «Eclipse Hopson» (23.06.2026); формулировки
> вычищены и параметризованы под Eclipse Library.
>
> **Фит для наших проектов:** готовый сценарный набор для **Educator-AI**
> (educator.eclipse-forge.ru) — встроить как пресет «Языковой репетитор» рядом с
> генерацией курсов; см. также [Open Culture](../README.md) и приём
> «Gemini делает курсы».

Плейсхолдеры: `[ЯЗЫК]` — изучаемый язык, `[УРОВЕНЬ]` — A1…C2, `[ТЕМА]` — тема урока,
`[ТЕКСТ]` — ваш фрагмент, `[СЛОВА]` — список слов/фраз.

---

## 1. Разговор с «носителем»

```
Ты — носитель [ЯЗЫК], живущий среди native-спикеров. Веди со мной естественный
живой диалог на тему «[ТЕМА]» на уровне [УРОВЕНЬ]. Говори как в жизни, не
по-учебному. После каждой моей реплики мягко исправляй явные ошибки
(грамматика, порядок слов, артикли) и коротко объясняй, как правильно.
Начни первым с вопроса по теме.
```

## 2. 30-минутный урок

```
Составь 30-минутный урок [ЯЗЫК] уровня [УРОВЕНЬ] по теме «[ТЕМА]»
(фокус: грамматика / разговорная практика / аудирование — выбери уместное).
Структура: 5 мин теория с примерами → 15 мин упражнения с моими ответами →
10 мин мини-тест из 10 вопросов с проверкой. Веди меня по шагам, жди ответов.
```

## 3. Карточки для запоминания

```
Преобразуй этот список в карточки для интервального повторения: [СЛОВА].
Для каждой карточки: слово на [ЯЗЫК] → перевод → пример в предложении →
короткий мнемонический приём для запоминания. Выдай таблицей,
готовой к импорту в Anki.
```

## 4. Разбор грамматических ошибок

```
Вот мой текст на [ЯЗЫК]: [ТЕКСТ].
Найди и отметь ВСЕ ошибки (грамматика, лексика, пунктуация, стиль).
Для каждой: что не так → как правильно → правило, которое стоит запомнить.
В конце — топ-3 ошибки, на которые мне обратить внимание в первую очередь.
```

## 5. Проверка прогресса (мок-экзамен)

```
Проведи пробный экзамен по [ЯЗЫК] уровня [УРОВЕНЬ]: 10 заданий (лексика,
грамматика, чтение, аудирование-имитация). Сначала выдай все вопросы и жди
моих ответов. После моих ответов покажи правильные, разбор каждого и итоговый
балл с рекомендацией, что подтянуть.
```

## 6. «Языковой движок» — погружение через перевод

```
Переведи этот текст на [ЯЗЫК]: [ТЕКСТ]. Затем по переводу задай мне 5 вопросов
на [ЯЗЫК]: на понимание смысла, на новую лексику и на устойчивые выражения.
Жди моих ответов и комментируй их.
```

---

**Как пользоваться:** сохрани диалог с нейросетью как «репетитора», занимайся в нём
ежедневно по 20–30 минут, чередуя промпты 1–6. Память диалога держит твой прогресс
и ошибки.
