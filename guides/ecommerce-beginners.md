# 🟢 Интернет-магазин с нуля — для чайников

> Это руководство для тех, кто **никогда** не делал интернет-магазин. Без программирования.
> Объясняем каждое слово, идём по шагам. Цель: сделать **первый рабочий магазин за вечер**, бесплатно.
> Когда освоишься и захочешь больше контроля → [гайд для разработчиков](ecommerce-dev.md).
> Каталог инструментов → [библиотека](../README.md#-разработка-интернет-магазинов) · сайт: <https://library.eclipse-forge.ru>

## 1. Что такое интернет-магазин (на пальцах)

Магазин в интернете — это сайт, где человек может **выбрать товар, положить в корзину и оплатить**.
Внутри он состоит из простых частей:

| Часть | Что это простыми словами |
|---|---|
| **Витрина** | То, что видит покупатель: страницы с товарами, картинки, цены |
| **Каталог** | Список твоих товаров (название, фото, цена, описание) |
| **Корзина** | «Тележка», куда покупатель складывает товары перед оплатой |
| **Оформление заказа (checkout)** | Форма: имя, адрес, телефон, способ оплаты и доставки |
| **Оплата** | Приём денег картой/СБП — через специальный сервис («платёжку») |
| **Доставка** | Как товар доедет: почта, СДЭК, самовывоз, курьер |
| **Админка** | Твой «пульт управления»: добавляешь товары, видишь заказы |

**Хорошая новость:** тебе НЕ нужно делать всё это руками. Есть готовые конструкторы,
где всё уже встроено — ты только добавляешь товары и нажимаешь кнопки.

## 2. Что понадобится (чек-лист новичка)

Перед стартом подготовь:

- [ ] **Товары** — что продаёшь (даже 1–3 для начала).
- [ ] **Фото товаров** — снятые на телефон сойдут, главное светло и без хлама в кадре.
- [ ] **Описания и цены** — короткий текст и цена для каждого товара.
- [ ] **Название магазина** — придумай простое и запоминающееся.
- [ ] **Способ принимать деньги** — об этом в разделе 5 (это проще, чем кажется).
- [ ] **2–3 часа времени** и желание. Денег на старте **не нужно** — начнём бесплатно.

## 3. Выбери свой уровень

Есть три пути. **Новичку — однозначно зелёный.**

| Уровень | Кому | Чем делаем | Сложность |
|---|---|---|---|
| 🟢 **Без кода** | Никогда не делал магазин | **Tilda** или **Ecwid** | Очень легко, по кликам |
| 🟡 **Готовый движок** | Хочешь больше возможностей, не боишься чуть покопаться | **WooCommerce** (на WordPress) | Средне |
| 🔴 **Свой код** | Ты разработчик, нужен полный контроль | Medusa + Next.js | Сложно → [dev-гайд](ecommerce-dev.md) |

Дальше — подробно по **зелёному пути на Tilda**. Это самый понятный старт.

## 4. 🟢 Магазин на Tilda за вечер — пошагово

[Tilda](https://tilda.cc) — конструктор сайтов: собираешь страницы из готовых блоков мышкой,
как конструктор Лего. У неё есть встроенный магазин. Есть бесплатный тариф для старта.

> Принцип везде одинаковый (в Ecwid, на других конструкторах — те же шаги).
> Поэтому, даже если выберешь другой сервис, логика та же.

**Шаг 1. Регистрация.** Зайди на [tilda.cc](https://tilda.cc), зарегистрируйся (почта + пароль).

**Шаг 2. Создай сайт.** Нажми «Создать новый сайт» → выбери шаблон с магазином
(ищи «Store» / «Магазин»). Шаблон — это готовый дизайн, который ты потом поменяешь под себя.

**Шаг 3. Включи каталог товаров.** В настройках сайта найди раздел **«Каталог товаров»** /
**Store**. Это твоя витрина с товарами.

**Шаг 4. Добавь товары.** В каталоге нажми «Добавить товар» и заполни:
- Название, цена, описание;
- загрузи фото;
- (по желанию) категория, наличие, варианты (размер/цвет).

Добавь 1–3 товара для начала — остальные позже.

**Шаг 5. Настрой корзину.** В большинстве шаблонов корзина уже есть. Проверь, что кнопка
«В корзину» появляется на товарах, а вверху есть значок корзины.

**Шаг 6. Подключи оплату.** В настройках магазина → **«Приём платежей»** выбери платёжку
(рекомендуем **ЮKassa** — см. раздел 5). Тебе дадут вставить два ключа из личного кабинета
платёжки — скопировал, вставил, готово. *Пока нет платёжки — можно временно поставить
«оплата при получении» или «по договорённости», чтобы запуститься сегодня.*

**Шаг 7. Настрой доставку.** В настройках → способы доставки: самовывоз, почта, СДЭК, курьер.
Для старта хватит «самовывоз» + «по договорённости».

**Шаг 8. Добавь обязательные страницы.** Сделай простые страницы (Tilda → новая страница):
- **Контакты** (как с тобой связаться);
- **Оферта** (правила покупки) и **Политика конфиденциальности** — без них нельзя принимать
  оплату по закону. Готовые шаблоны текста ищутся за минуту в поиске; вставь и подставь свои данные.

**Шаг 9. Опубликуй.** Нажми **«Опубликовать»**. Сайт станет доступен по адресу вида
`твой-проект.tilda.ws`. Хочешь красивый адрес — подключи свой домен (раздел 6).

**Шаг 10. Проверь сам.** Зайди как покупатель: выбери товар → корзина → оформи тестовый заказ.
Убедись, что заказ пришёл тебе в админку. **Готово — у тебя работает магазин!**

## 5. Как принимать деньги — на пальцах

Чтобы магазин принимал оплату картой, нужен **эквайринг** — это сервис-посредник
(«платёжка»), который проводит деньги от покупателя к тебе. Сам сайт деньги не «ловит».

**Что выбрать новичку: [ЮKassa](https://yookassa.ru)** — самый ходовой в РФ, есть готовая
интеграция почти со всеми конструкторами. Альтернативы: [Тинькофф Касса](https://www.tbank.ru/kassa/),
[CloudPayments](https://cloudpayments.ru), [Robokassa](https://robokassa.com).

**Как подключить (общая схема):**
1. Зарегистрируйся в ЮKassa, заведи «магазин» в их кабинете.
2. Тебе дадут **shopId** и **секретный ключ** — это как логин/пароль для приёма денег.
3. В настройках своего магазина (Tilda → приём платежей) вставь эти два значения.
4. Сделай тестовую оплату — деньги должны прийти.

**Важно:** чтобы принимать деньги легально, нужен статус — см. раздел 6.

## 6. Юридический минимум (не страшно)

Чтобы принимать оплату честно, нужен один из статусов:

- **Самозанятость** — самый простой для старта. Оформляется за 10 минут в приложении
  **«Мой налог»** (ФНС), бесплатно, налог 4–6% только с продаж. Подходит, если продаёшь
  своё/услуги и оборот небольшой.
- **ИП** — если оборот больше или нужны сотрудники/опт. Чуть сложнее, но открывается онлайн.

**Обязательные страницы на сайте** (без них платёжка не подключит оплату):
- **Оферта** (договор-условия покупки);
- **Политика конфиденциальности** (как обрабатываешь данные — 152-ФЗ);
- **Контакты** и условия **возврата/доставки**.

Шаблоны этих текстов бесплатно гуглятся за минуту — бери, подставляй свои реквизиты.

**Домен** (раздел 6½): `твой-магазин.ru` — это адрес сайта. Можно купить за ~100–200 ₽/год
у регистратора. Или на старте — бесплатный поддомен конструктора / наш `*.eclipse-forge.ru`.

## 7. Страхи новичка → честные ответы

**«Я не программист — справлюсь?»** Да. На Tilda/Ecwid код не нужен вообще, всё мышкой.

**«Нужны ли деньги на старте?»** Нет. Конструктор — бесплатный тариф, статус самозанятого —
бесплатно, тексты — бесплатно. Платишь только когда пойдут продажи (комиссия платёжки ~2–3%).

**«А если не пойдёт?»** Ничего не теряешь — просто перестаёшь публиковать. Поэтому начни
с малого: 2–3 товара, проверь спрос, потом расширяй.

**«Нужен ли склад/много товара?»** Нет. Можно продавать под заказ, цифровое, услуги, хендмейд.

**«Чем это отличается от того, что делают разработчики?»** Конструктор — быстро и просто,
но меньше контроля над дизайном/логикой. Когда магазин вырастет — переходишь на свой код
(WooCommerce → потом headless). Об этом — в [гайде для разработчиков](ecommerce-dev.md).

## 8. Чеклист «магазин готов к запуску»

- [ ] Есть 1+ товар с фото, ценой, описанием
- [ ] Работает корзина и оформление заказа
- [ ] Подключена оплата (или временно «при получении»)
- [ ] Настроена доставка (хотя бы самовывоз)
- [ ] Есть страницы: контакты, оферта, политика, возврат
- [ ] Оформлен статус (самозанятость/ИП)
- [ ] Сам сделал тестовый заказ — всё пришло
- [ ] Сайт опубликован, адрес работает

Поставил все галочки — **поздравляю, ты сделал интернет-магазин!** 🎉
