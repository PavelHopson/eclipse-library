# July 2026 learning / security / product radar

## Eclipse Chat

- **InsightsLM** - reference для knowledge rooms: документы, обсуждение, summary и source-grounded answers в одном месте.
- **Council of High Intelligence** - не копировать "18 мудрецов", а взять protocol: options, risks, dissent, decision log.
- **CS Video Courses** - training rooms для команды и клиентов: учебные подборки по инженерным темам.

## Eclipse AI Hub

- **InsightsLM** - self-hosted NotebookLM-like RAG benchmark.
- **GFusion / ZCode / GLM** - provider/model radar, не production default без evals.
