# 📸 Настройки камеры iPhone для контента

> Лучшие настройки съёмки фото/видео на iPhone для визуального контента
> (соцсети, лендинги, демо проектов). Источник: дроп «Eclipse Hopson» (24.06.2026).
>
> **Фит:** контент для **Shotforge** (демо-кадры), маркетинг/соцсети проектов
> Eclipse Forge, превью лендингов. Снято на телефон → меньше зависимость от
> стоковых фото (которые Pavel не любит).

## Видео — `Настройки → Камера → Запись видео`

| Параметр | Значение | Почему |
|---|---|---|
| Разрешение/частота | **4K / 60 fps** (мало памяти → 1080p/60) | Чёткость + плавность; 1080p/60 норм для соцсетей |
| HDR-видео | **Выключить** | На монтаже даёт перепады яркости; красиво только на экране телефона |
| Автовыбор fps | **Выключить** | Иначе iPhone сам режет кадры → «мыло» в сложных сценах |
| Улучшенная стабилизация | **Включить** | Меньше тряски |
| Блокировка баланса белого | **Включить** | Цвет не «плывёт» при смене освещения |

## Фото — `Настройки → Камера → Форматы`

| Параметр | Значение | Почему |
|---|---|---|
| Формат захвата | **Высокоэффективный (HEIF)** | Меньше вес без потери качества |
| Режим фото | **24 Мп** | Баланс качество/размер |
| ProRAW | **JPEG Lossless** | Максимум данных для постобработки без потерь |

## Интерфейс — `Настройки → Камера`

- **Сетка** и **Уровень** — включить (правило третей + не завалить горизонт).
- **Автомакро**, **коррекция объектива**, **портреты в режиме фото** — включить.
- **«Сохранение настроек»** — активировать все пункты, иначе камера откатывается
  к дефолтам при каждом запуске.
